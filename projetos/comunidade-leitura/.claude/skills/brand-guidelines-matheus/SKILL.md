# /brand-guidelines-matheus

## Princípio central

Antes de gerar qualquer conteúdo visual ou textual para @omatheusfariaa, ler
`brand.md` e aplicar todas as diretrizes.

O conteúdo só está correto quando passa no teste:
> "Parece que o Matheus está falando diretamente comigo — não parece IA, não parece guru."

---

## Passo 1 — Carregar as diretrizes

Ler `brand.md` e carregar:
- Paleta de cores (hex exatos)
- Tipografia (handwriting para destaque, sans-serif para texto)
- Elementos visuais (anotações à mão, fotos reais, overlay)
- Tom de voz (o que dizer e o que nunca dizer)
- Referências por formato

---

## Passo 2 — Checklist antes de entregar

Para cada output, verificar:

**Visual:**
- [ ] Usa cores da paleta (sem branco puro, sem preto puro, sem neon)?
- [ ] Tipografia handwriting nos destaques?
- [ ] Tem pelo menos um elemento "anotado à mão" (seta, sublinhado, círculo)?
- [ ] Foto real ou fundo terroso (não gradiente genérico)?

**Texto:**
- [ ] A frase mais forte está na primeira linha?
- [ ] Fala pra uma pessoa (não "vocês" ou "todos")?
- [ ] Tem alguma palavra de self-help clichê? → remover
- [ ] Tem construção de IA ("é importante", "neste contexto")? → remover
- [ ] O CTA é específico (não "me segue" ou "curte")?

---

## Passo 3 — Aplicação por formato

### Carrossel
Seguir `/carrossel` + aplicar paleta bege/terracota + handwriting na capa.
Referência visual: @escuta.ela

### Roteiro / post de feed
Tom do Faria: convicto, presente, fala pra uma pessoa.
Nunca soar como resumo do livro — sempre como experiência vivida.

### Link na bio / página HTML
Tons terrosos, tipografia limpa, sem gradiente.
O Matheus como elemento central.

### Descrição de vídeo / legenda
Puxar cor do vídeo para o texto (quando possível).
Inspiração: Fê Alves.
