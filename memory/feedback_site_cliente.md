A Assessoria Raiz cria sites completos do zero para os clientes quando necessário, como fez para a CZ Contabilize (czcontabilize.com.br). Não assumir que site de cliente é externo ou apenas "ajustado" — pode ser uma entrega da própria Raiz.

**Why:** Matheus corrigiu duas vezes essa suposição errada.
**How to apply:** Antes de comentar sobre o site de um cliente, verificar no briefing/contexto se a Raiz é responsável pelo site.
