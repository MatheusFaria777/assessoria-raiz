# Casa Valério

## O que é
Restaurante (nicho alimentação/delivery). Cliente de tráfego pago com produção de conteúdo visual inclusa (vídeos, reels).

## Tipo
Cliente externo

## Serviços
- Meta Ads (campanhas de Vendas + Distribuição)
- Produção de conteúdo visual (vídeos e reels)
- Roteiros de vídeo baseados no tom de voz do cliente

## Arquivos importantes
- `briefing.md` — contexto do cliente, público, posicionamento
- `contexto.md` — histórico de campanhas, métricas mensais, aprendizados
- `tom-de-voz.md` — guia de tom de voz extraído dos vídeos do cliente (gerar via Whisper)
- `videos/` — MP4s do cliente + `transcricoes.md` (transcrições geradas pelo Whisper)
- `planilha` — https://docs.google.com/spreadsheets/d/1UkssfV0gIhrLvF4qaXHGBJW_llTrikkWImJD-5gi3pU

## Regras específicas
- Ao criar roteiros ou copies, sempre calibrar pelo tom de voz extraído dos vídeos do cliente
- Pedir vídeos gravados pelo cliente quando precisar capturar a forma como ele fala
