# Feedback e Aprendizados — Assessoria Raiz

## O que funciona nos criativos
- Vídeos com ângulo de dor específica do dono de loja ("carro parado é dinheiro parado")
- Prova social com nome de cliente real + número concreto (6 carros, 7 dias)
- CTA de diagnóstico gratuito — baixa fricção, fácil de clicar

## O que não funcionou
- **Formulário Respondi como destino:** carregamento lento mata a conversão mesmo com CTR alto
- Campanhas sem tracking adequado (Respondi sem pixel) — impossível otimizar sem dados

## Regras para próximas campanhas
- Sempre usar formulário instantâneo do Meta ou LP com pixel configurado
- Nunca subir formulário externo sem testar velocidade de carregamento antes
- Ao criar nova campanha, testar pelo menos 3 vídeos diferentes no mesmo ad set
