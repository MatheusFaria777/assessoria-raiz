Política de Privacidade

Assessoria Raiz

Última atualização: 19 de maio de 2026

1. Quem somos

A Assessoria Raiz é uma assessoria de tráfego pago que gerencia campanhas de Meta Ads (Facebook e Instagram) e Google Ads em nome de seus clientes. Nosso aplicativo interno acessa a API de Marketing da Meta exclusivamente para operar campanhas autorizadas por cada cliente.

2. Quais dados acessamos

Por meio da API de Marketing da Meta, acessamos:

  • Dados de contas de anúncio autorizadas (campanhas, conjuntos de anúncios, criativos)
  • Métricas e insights de desempenho das campanhas
  • Informações de páginas do Facebook vinculadas às contas gerenciadas
  • Dados de público e segmentação configurados nas campanhas

Não coletamos dados pessoais de usuários finais (visitantes dos anúncios). Todos os dados acessados se referem às contas dos nossos clientes, com autorização expressa.

3. Como usamos os dados

  • Criação, edição e monitoramento de campanhas publicitárias
  • Geração de relatórios de performance para os clientes
  • Otimização de orçamentos e segmentações
  • Automação de processos internos de gestão de anúncios

Os dados não são utilizados para nenhuma finalidade além das descritas acima.

4. Compartilhamento de dados

Os dados acessados via API da Meta são utilizados exclusivamente pela equipe da Assessoria Raiz para as finalidades descritas nesta política. Não vendemos, alugamos nem compartilhamos dados com terceiros.

Utilizamos a API oficial da Meta Platforms, sujeita aos Termos de Serviço para Desenvolvedores da Meta.

5. Armazenamento e segurança

Tokens de acesso e credenciais são armazenados de forma segura em variáveis de ambiente, nunca em código-fonte ou sistemas de controle de versão públicos. O acesso é restrito à equipe interna da Assessoria Raiz.

Dados de desempenho de campanhas são retidos pelo período necessário para geração de relatórios históricos, respeitando as políticas de retenção da Meta.

6. Direitos dos titulares

Nossos clientes podem, a qualquer momento, revogar o acesso da Assessoria Raiz às suas contas de anúncio diretamente no painel do Meta Business Manager, ou entrando em contato conosco pelo e-mail abaixo.

7. Cookies e rastreamento

Esta página de política de privacidade não utiliza cookies de rastreamento nem ferramentas de analytics de terceiros.
